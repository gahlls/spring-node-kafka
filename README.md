# spring-node-kafka

# Dependências
<ul>
  <li>Node.js >= 8.0.0</li>
  <li>Yarn</li>
  <li>Docker</li>
  <li>Java >= 8</li>
</ul>

# Iniciando
1. Rode <code>docker-compose up</code> dentro da raiz do projeto.
2. Rode <code>yarn</code> dentro da pasta consumer, para instalar as dependências do node.
3. Rode o spring-boot para iniciar o producer
